## 如何引入
Step 1. 添加 JitPack repository 
```
allprojects {
	repositories {
		...
		maven { url "https://jitpack.io" }
	}
}
```
Step 2. 添加 Gradle依赖
```
dependencies {
    ...
    implementation 'com.github.loperSeven:tablayout-ext:$version'//具体版本请看顶部jitpack标识，仅支持androidx
}


```
